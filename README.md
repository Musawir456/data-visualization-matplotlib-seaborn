# Data Visualization with Matplotlib and Seaborn

This repository focuses on **data visualization** using Python libraries **Matplotlib** and **Seaborn**, which are essential for exploring and presenting data insights.

## Topics Covered
- Introduction to Matplotlib
- Plot types: Line, Bar, Histogram, Scatter, Pie
- Customizing plots: Titles, Labels, Legends, Colors
- Subplots and figure size adjustments
- Introduction to Seaborn
- Plot types: Scatterplot, Boxplot, Histogram, Heatmap, Pairplot
- Styling and color palettes in Seaborn
- Combining Matplotlib and Seaborn for advanced visualization
- Practical examples for exploratory data analysis (EDA)

## Tools & Libraries
- Python
- Matplotlib
- Seaborn
- NumPy
- Pandas
- Jupyter Notebook

## Purpose
- Learn to visualize data effectively
- Practice creating charts for analysis and presentations
- Enhance data storytelling skills
- Prepare data for reports, dashboards, and machine learning insights
